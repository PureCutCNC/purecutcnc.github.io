Choose a Track

Stable or snapshot

Stable is the safer install for regular use. Snapshot is the newest in-progress desktop build and is better for testing recent changes before the next full release.

  • Choose stable if you want the most conservative desktop build.
  • Choose snapshot if you are validating current work or checking a recent fix.
  • If a platform/channel has no package yet, that card will stay unavailable until the next build is published.
No Install

Browser app still works

If you just want to test features quickly, the web build is still the fastest path. Desktop packages are mainly for platform-specific validation and local install workflows.
